# Number Guessing Game

Practicing making functions, while loop, if-statements, and using the `random library`. 

- The random library will generate an integer within a range that is set by the user.
- The program will ask the start range and end range, but if the user ever decides to answer incorrectly, the if-statement

``python 
if x>y:
  print("Invalid range")
  return
```
- will return and go ask the question again
- A while loop will be activated and it will compare the input from the user if it's too high or too low
- And will continue to add the number of guesses, and compare until the user gets it right
